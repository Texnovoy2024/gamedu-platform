# 🎮 GamEdu - Gamifikatsiyalangan Ta'lim Platformasi

![GamEdu Logo](public/logo.png)

## 📖 Loyiha haqida

GamEdu - bu XP, darajalar, badge va reytinglar orqali o'quvchilarni motivatsiyalovchi zamonaviy gamifikatsiyalangan ta'lim platformasi. O'qituvchilar topshiriqlar yaratadi, o'quvchilar esa ularni bajarib, XP to'playdi va reytingda yuqoriga chiqadi.

## ✨ Asosiy Xususiyatlar

### 👨‍🏫 O'qituvchilar uchun:
- ✅ Turli xil topshiriqlar yaratish (ko'p variantli, to'g'ri/noto'g'ri, bo'sh joy, anagram, va boshqalar)
- 📊 O'quvchilar progressini kuzatish
- 📈 Statistika va reyting ko'rish
- 🎯 XP va qiyinlik darajasini sozlash
- 📄 PDF/DOCX fayllardan savollarni import qilish

### 👨‍🎓 O'quvchilar uchun:
- 🎮 10+ ta mini o'yin (Bayroq tanish, Qo'shiq topish, Xarita o'yini, va boshqalar)
- ⭐ XP to'plash va daraja oshirish
- 🏆 Badge va yutuqlar olish
- 📊 Reyting jadvalida raqobatlashish
- 🔥 Streak tizimi (ketma-ket kunlar)
- 💰 Tanga to'plash tizimi
- 📅 Kunlik vazifalar

## 🛠️ Texnologiyalar

- **Frontend**: React 18 + TypeScript
- **Styling**: Tailwind CSS
- **Routing**: React Router v6
- **Animations**: Framer Motion
- **Icons**: Lucide React
- **Build Tool**: Vite
- **Storage**: LocalStorage (client-side)

## 🚀 O'rnatish va Ishga Tushirish

### Talablar:
- Node.js 18+ 
- npm yoki yarn

### O'rnatish:

```bash
# Repository ni clone qiling
git clone https://github.com/Texnovoy2024/gamedu-platform.git

# Papkaga kiring
cd gamedu-platform

# Dependencies ni o'rnating
npm install

# Development server ni ishga tushiring
npm run dev
```

Brauzerda `http://localhost:5173` ochiladi.

### Build qilish:

```bash
npm run build
```

Build fayllari `dist` papkasida bo'ladi.
